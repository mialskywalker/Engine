# Engine

Engine built for the Second Assignment for the UPC Masters Degree.

Link to [GitHub Repository](https://github.com/mialskywalker/Engine)

## How to run
1. Download the release from the GitHub repository.
2. Unzip the release package into your directory.
3. Open the `Engine` folder and run `Engine.exe`.

## Controls
- W/A/S/D keys: Move the camera forward, backward, left or right
- Hold RIGHT mouse button: Free-look (rotate camera view)
- Hold Left ALT key + LEFT mouse button: Orbit object
- Hold SHIFT key: Doubles camera's movement speed
- F key: Focuses the camera on the game object

## Object Manipulation
- W key: Translate object
- E key: Rotate object
- R key: Scale object

## Notes
Includes only runtime-required assets:
- Executable
- Compiled shaders (.cso)
- Game assets
- Required DLLs
Click on Reset Transform to scale down the model to 0.01 in order to view properly
